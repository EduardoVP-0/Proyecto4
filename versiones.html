SERVIDOR WEB APACHE

Versión Estable Actual

La versión estable más reciente es Apache 2.4.57, lanzada en octubre de 2024. Esta versión incluye correcciones de seguridad y algunas mejoras de rendimiento.

Principales Versiones Anteriores

APACHE 2.0

Esta versión introdujo mejoras en el rendimiento y la escalabilidad, así como soporte para múltiples plataformas y un modelo de procesamiento más eficiente.

APACHE 2.2

Lanzada en diciembre de 2005, esta versión trajo consigo nuevas características como el soporte para módulos de autenticación más avanzados y mejoras en la configuración de virtual hosts. Se recomienda a los usuarios de la rama 2.2 actualizar a la versión 2.4 para beneficiarse de nuevas funcionalidades y mejoras de seguridad.

APACHE 2.4

Esta es la versión más utilizada actualmente y se lanzó en febrero de 2012. Introdujo un nuevo modelo de procesamiento de solicitudes, mejoras en la gestión de la memoria y un mejor soporte para configuraciones de alto rendimiento. La versión 2.4.43 o superior es necesaria para operar un servidor web TLS 1.3 con OpenSSL 1.1.1.

APACHE 2.5

Aunque aún no es una versión estable, se están realizando desarrollos en esta rama, que promete incluir más mejoras y características avanzadas.

Requerimientos de Sistema Operativo

Linux


La mayoría de las distribuciones populares como Ubuntu, Debian, CentOS y Red Hat son compatibles con Apache.

Windows


Apache también puede instalarse en sistemas Windows (a partir de Windows 7), aunque Linux suele ser la opción preferida en servidores de producción.

macOs


Compatible, pero menos común en entornos de producción.

Requerimientos de Hardware

RAM


Se recomienda un mínimo de 512 MB de RAM para pequeñas cargas de trabajo, pero para un servidor de producción se recomienda al menos 1 GB de RAM. Para sitios con mucho tráfico, 2 GB o más serían adecuados.

CPU


Un procesador de 1 GHz es suficiente para la mayoría de los entornos de prueba o sitios pequeños. Para cargas de trabajo pesadas, se recomiendan múltiples núcleos.
